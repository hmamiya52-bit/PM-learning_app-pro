# F2-figures ノート図表追加キュー

> 作成: Claude（2026-05-17）
> 使用方法: 共通テンプレ `tasks/codex/F2-figures_template.md` と併用。Codex は「**未着手**」最上位の項目を1〜3個取って作業する。
> 方針: ノート内のマトリクス・モデル・概念図を SVG/table で視覚化し、暗記効率と理解度を向上させる。

## 状態の凡例
- **未着手**: 未だ作業されていない
- **進行中（Codex / YYYY-MM-DD）**: Codex が作業開始
- **完了（YYYY-MM-DD / commit <hash>）**: Codex 図表追加完了
- **承認済**: ユーザレビューで OK

## 補足

### Codex 1セッションあたりの処理範囲
1〜3 セクション可。各セクションが独立しているためコンフリクトリスクが低い。ただし複数を扱う場合、各セクションの設計時間を確保すること。

### Claude 並行作業
Codex が図表追加中、Claude はノート本文生成（F2-P1 残カテゴリ）を並行可能。`src/pages/NoteDetail.tsx` の編集箇所が**異なるカテゴリ**であれば並列で進められる。

### 終了条件
全19項目が「完了」状態になればフェーズ目標達成。優先1〜8（stakeholder + team）の完了で **MVP** とみなしてよい。
