### 3. 83d9827: 達成度レポート（MorningAchievementReport.tsx 新規 295 行）

#### 構造
- モーダル形式（`role="dialog"` + `aria-modal="true"`、Escape で閉じる）
- 年度（行）× 問番号 1〜25（列）のヒートマップ表
- 5 状態色分け: 連続正解=金 `2○` / 正解=緑 `○` / 不正解=橙 `×` / 連続不正解=赤 `×2` / 未解答=空
- 各セルクリックで `startSession([question], 'single', ...)` → 1 問演習画面へ遷移
- フッタ凡例 + 件数集計
- サマリ: 「全 300 問中 N 問解答済み (P%)」+ 進捗バー

#### a11y
- `aria-labelledby` / `aria-modal` 設定済
- 各セルボタンに `title` + `aria-label`（年度・問番号・状態・最終解答日を含む文字列）
- 凡例から状態の意味を視覚的に学習可能
- Escape キーで閉じる

#### 実機検証
- モバイル（375px）: 表は `min-w-[920px]` で横スクロール、sticky 年度列・問番号ヘッダで方向感維持 ✓
- デスクトップ: `max-w-6xl` で 1 画面に収まる想定 ✓
- 凡例フッタにすべての状態 + 該当件数 ✓
- 1 問演習への遷移動作確認: 達成度レポート → セルクリック → `/morning/session` で IPA 公式問題が表示 ✓
