# F1.5-P4 Codex OCR 作業メモ

## 実施内容

- IPA 公式 PDF を取得し、令和6年度 秋期 プロジェクトマネージャ試験 午前II 25問を `src/data/officialMorningQuestions.ts` に投入した。
- 既存の placeholder 3問は削除し、`om-R6-1` 〜 `om-R6-25` の25件に差し替えた。
- `explanation` は全問 `''` のまま。解説作成は次工程 Claude 担当。
- `sourceUrl` は全問、問題冊子 PDF URL に統一した。

## 使用ツール

- PDF 取得: `Invoke-WebRequest`
- ページ画像化: PyMuPDF (`fitz`) で 4x scale PNG 出力
- OCR:
  - Tesseract OCR 5.5.1 (UB Mannheim build)
  - `jpn+eng` tessdata fast / best
  - Windows OCR (`winsdk.windows.media.ocr`)
- 照合:
  - IPA 原 PDF 画像をページ単位で目視確認
  - OCR 差分確認用に公開過去問ページを補助参照

## OCR 誤認識と補正メモ

- 問1: `KRY` → `成果物`、`ブロジェクト` → `プロジェクト` などを原画像で補正。
- 問2: RACI 選択肢の選択肢記号・`説明責任` の誤認識を補正。
- 問4: 図の値 `BAC=100 / PV=50 / AC=40 / EV=34` を画像から確認し、本文注記に追加。
- 問5: アローダイアグラムは UI 画像非対応のため、作業 A～I とダミー作業を本文注記に追加。
- 問6: `A～D`、`PDM`、`FS`、所要日数、試験設備条件を画像で確認。
- 問9: 指数表記は原 PDF の上付き表現を `E=5.2L^0.98` としてプレーンテキスト化。
- 問10: 分数式は UI 表示を優先し、`1／X＋1／Y＋1／Z`、`1／(1／X＋1／Y＋1／Z)` のようにプレーンテキスト化。
- 問12: 開発要員投入計画表は画像から数値を確認し、本文に月別テキストとして注記。
- 問15: `CMMI モデル V2.0` を画像で確認。
- 問20: `実費償還契約` の OCR 欠落を補正。
- 問23: `脅威情報` の誤認識を補正。
- 問25: `ワークアラウンド`、`脆弱性修正プログラム` を画像で確認。

## 図表注記

- 図表・数式を含む問題として、問4・問5・問6・問10・問12 の `categoryId` に `(figure)` suffix を付与した。
- 現行 UI は問題画像レンダリング非対応のため、問4・問5・問6・問12 は `questionText` に図表の読み取り内容を補足した。
