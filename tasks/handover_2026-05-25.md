## 【教訓・注意点】

### マークアップ規約（最重要）

午前II 解説と NoteDetail で共通マークアップ:
- `==X==`: 赤字太字（重要語句・暗記対象）
- `__X__`: ネイビー（#9d5b8b）太字（構造ラベル・分類軸・「ア/イ/ウ/エ」）

**`==X__` / `__X==` 開閉ミスマッチを避ける**:
- 過去セッションで 100 件以上のミスマッチを修正済
- 投入後は必ず**厳密チェック**を行う（総数偶数では検出漏れする）
- 厳密チェックの実装パターン（必要なら一時スクリプトで使用）:

```javascript
const OPEN = String.fromCharCode(1)
const CLOSE = String.fromCharCode(2)
let buf = exp
buf = buf.replace(/__([^_=\\]{1,30}?)__/g, `${OPEN}$1${CLOSE}`)
const eqUs = buf.match(/==[^=\\]{1,200}?__/g)
const usEq = buf.match(/__[^=\\]{1,200}?==/g)
if (eqUs || usEq) { /* ミスマッチ検出 */ }
```

### 機械修正スクリプトの保護パターン

`==X__` を `==X==` に直す機械修正で正常な `__X__` を壊さないよう、制御文字（SOH/STX）で保護してから置換:

```javascript
buf = buf.replace(/__([^_=\\]{1,30}?)__/g, `${OPEN}$1${CLOSE}`)
buf = buf.replace(/==([^=\\]{1,200}?)__/g, '==$1==')
buf = buf.replace(/__([^=\\]{1,200}?)==/g, '==$1==')
buf = buf.replace(new RegExp(`${OPEN}([^${CLOSE}]+?)${CLOSE}`, 'g'), '__$1__')
```

### MathText 拡張

`src/components/MathText.tsx` は数式マークアップ（`frac{}{}` / `^{}` / `_{}`）に加えて `==X==` / `__X__` をサポート済み。

### スクリプト内の制御文字

スクリプトに直接制御文字を書くとファイル保存時に消える場合があるので、必ず `String.fromCharCode(1)` で生成する。
