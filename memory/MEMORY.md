# PM Learning App — プロジェクト共有メモ

リポジトリにcommitされるチーム共有メモのINDEX。
個人用 auto memory（Claudeセッション固有・`~/.claude/projects/...`）とは別物。

## ファイル一覧

- [リスクレジスタ](risks.md) — IPA著作権・コンテンツ品質・PMBOK改訂など8件のリスク管理

## 運用ルール

- メモ更新時はコミットprefix `[Doc]` を使用
- リスクレジスタはフェーズ移行時（F1→F1.5、F1.5→F2）に再評価
- 個人プロファイル・嗜好情報はここには置かない（auto memory側で管理）
- 設計判断の根拠は `detailed_design.md` 本体に記載、ここには方針サマリのみ
